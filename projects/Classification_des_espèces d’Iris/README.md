# Classification des espèces d’Iris (IA / Data)

## Description
Ce projet porte sur l’application de techniques de Machine Learning pour la classification des espèces de fleurs Iris à partir de leurs caractéristiques morphologiques.

## Objectifs
- Mettre en œuvre un algorithme d’apprentissage automatique
- Analyser et préparer un jeu de données
- Évaluer les performances d’un modèle de classification
- Visualiser les résultats et les données

## Méthodologie
- Nettoyage et préparation des données
- Analyse exploratoire
- Entraînement d’un modèle K-Nearest Neighbors (KNN)
- Évaluation et visualisation des résultats

## Technologies utilisées
- Python
- Pandas, NumPy
- Scikit-Learn
- Matplotlib

## Compétences développées
- Analyse et traitement de données
- Apprentissage supervisé
- Modélisation et visualisation
- Utilisation de bibliothèques IA / Data
