# Système d’accès automatisé par RFID pour salles de sport

## Description
Ce projet consiste en la conception et le développement d’un système de contrôle d’accès automatisé pour des salles de sport, basé sur l’utilisation de cartes RFID. Il permet de gérer les accès des utilisateurs de manière sécurisée et centralisée.

## Objectifs
- Automatiser le contrôle des entrées via des cartes RFID
- Gérer les utilisateurs et leurs autorisations
- Mettre en place une communication réseau fiable entre les composants
- Stocker et exploiter les données d’accès dans une base de données

## Fonctionnalités principales
- Identification des utilisateurs par carte RFID
- Enregistrement et gestion des accès
- Interface web pour la consultation et l’administration
- Communication réseau via le protocole TCP

## Technologies utilisées
- Base de données : PostgreSQL  
- Interface web : HTML / CSS  
- Programmation : Python, Java  
- Réseaux : TCP
