# Parcours actuel

## SOC Level 1 — TryHackMe

Le Security Operations Center (SOC) est l’équipe chargée de surveiller, détecter et répondre aux incidents de sécurité dans une organisation.

Un SOC Analyst est notamment responsable de :

- l’analyse des logs  
- la détection d’activités suspectes  
- l’analyse d’alertes de sécurité  
- l’investigation d’incidents  
- la collaboration avec les équipes de sécurité  

Ce parcours permet d’acquérir les compétences techniques nécessaires pour un poste junior en Blue Team.


---

# Compétences étudiées dans ce parcours

Le parcours couvre plusieurs domaines essentiels de la cybersécurité défensive :

## Introduction à la Blue Team

- rôle du SOC  
- fonctionnement d’un centre opérationnel de sécurité  
- missions d’un SOC Analyst  

---

## Outils du SOC

Découverte des solutions utilisées dans les SOC :

- **SIEM** (Security Information and Event Management)  
- **EDR** (Endpoint Detection and Response)  
- **SOAR** (Security Orchestration Automation and Response)  

---

## Frameworks de cyberdéfense

Étude des modèles utilisés pour comprendre les attaques :

- **MITRE ATT&CK**  
- **Cyber Kill Chain**  
- **Pyramid of Pain**  

---

## Analyse de phishing

Apprentissage de techniques pour :

- analyser des emails suspects  
- identifier des tentatives de phishing  
- examiner les en-têtes d’emails  
- détecter les indicateurs de compromission  

---

## Analyse du trafic réseau

Utilisation d’outils comme Wireshark pour :

- analyser le trafic réseau  
- détecter des activités malveillantes  
- comprendre les attaques réseau  

---

## Détection d’intrusion

Utilisation d’**IDS** comme **Snort** pour :

- analyser le trafic réseau  
- détecter des anomalies  
- identifier des menaces en temps réel  

---

## Monitoring des systèmes

Analyse des logs de sécurité :

- Windows Security Monitoring 
- Linux Security Monitoring 

Objectif : détecter des attaques et comportements suspects sur les systèmes.

---

## Concepts de malware

Comprendre :

- les différents types de malware  
- leurs objectifs  
- leurs méthodes d’infection  
- les techniques modernes d’attaque  

---

## Threat Intelligence

Utilisation de sources de renseignement pour :

- enrichir les alertes  
- analyser les menaces  
- améliorer les investigations
