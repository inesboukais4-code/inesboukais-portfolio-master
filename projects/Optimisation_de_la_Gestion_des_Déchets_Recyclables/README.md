# Optimisation de la Gestion des Déchets Recyclables

## Description
Ce projet, réalisé dans le cadre d’un mémoire, vise à optimiser la gestion des déchets recyclables à travers la modélisation des flux et le développement d’une solution web permettant la gestion et la visualisation des données.

## Objectifs
- Analyser et modéliser les flux de déchets recyclables
- Proposer une solution technique pour améliorer la valorisation des matériaux
- Concevoir une application web pour la gestion des données

## Travaux réalisés
- Modélisation des processus et des flux
- Conception et développement d’une application Web Full-Stack
- Structuration et exploitation des données

## Technologies utilisées
- Front-end : HTML / CSS / JavaScript
- Back-end : PHP
- Base de données : MySQL

## Compétences développées
- Modélisation et analyse de systèmes
- Développement Web Full-Stack
- Gestion de bases de données
- Rédaction et structuration d’un projet académique
