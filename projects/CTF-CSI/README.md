# Capture The Flag (CTF) – CSI UMMTO

## Résolution de challenges en cybersécurité

Ce projet présente ma participation à un Capture The Flag (CTF) organisé par CSI UMMTO, où j’ai travaillé sur plusieurs challenges de cybersécurité dans différents domaines.

Les CTF sont des compétitions pratiques permettant de développer des compétences en analyse de sécurité, résolution de problèmes et investigation technique.

---

# Objectifs du projet

Les objectifs principaux de cette participation étaient :

- développer des compétences pratiques en cybersécurité
- résoudre des challenges dans plusieurs domaines techniques
- améliorer mes capacités d’analyse et de recherche
- appliquer des concepts théoriques dans des scénarios réels

Les CTF permettent de se confronter à des problèmes similaires à ceux rencontrés dans les métiers de la cybersécurité.

---

# Domaines abordés

Durant ce CTF, plusieurs catégories de challenges ont été explorées :

## Cryptographie

Les challenges de cryptographie consistent à analyser et casser des systèmes de chiffrement.

Compétences travaillées :

- analyse de messages chiffrés
- identification d’algorithmes cryptographiques
- décryptage à l’aide d’outils ou de scripts
- compréhension des faiblesses cryptographiques

---

## Forensic

Les challenges de forensic (analyse forensique) consistent à analyser des fichiers ou systèmes afin de retrouver des informations cachées.

Compétences travaillées :

- analyse de fichiers
- récupération d’informations cachées
- analyse de logs
- investigation numérique

---

## Web Security

Les challenges web portent sur l’analyse d’applications web vulnérables.

Compétences travaillées :

- identification de vulnérabilités web
- analyse du code HTML et JavaScript
- manipulation de requêtes HTTP
- exploitation de failles web simples

---

## Reverse Engineering

Les challenges de reverse engineering consistent à analyser des programmes afin de comprendre leur fonctionnement.

Compétences travaillées :

- analyse de binaires
- compréhension du comportement d’un programme
- recherche de mécanismes de protection
- identification de la logique interne d’une application

---

# Compétences développées

Cette expérience m’a permis de développer plusieurs compétences importantes :

- raisonnement logique et résolution de problèmes
- analyse de sécurité
- recherche et exploitation d’informations techniques
- travail sous contrainte de temps
- collaboration et partage de connaissances

---

# Outils utilisés

Selon les challenges, différents outils ont été utilisés :

- CyberChef
- Wireshark
- Burp Suite
- Ghidra
- strings / binwalk
- navigateurs web et outils développeur

---

# Ce que ce projet m’a apporté

Cette participation à un CTF m’a permis de :

- découvrir différents domaines de la cybersécurité
- pratiquer la résolution de challenges techniques
- renforcer mes compétences en analyse et investigation
- développer une approche méthodique face aux problèmes

Les CTF sont un excellent moyen d’apprendre la cybersécurité de manière pratique et interactive.
